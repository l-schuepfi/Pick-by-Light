# Pick-by-Light
Programm starten:

- `source venv/bin/activate`
- `sudo pigpiod`
- `sudo $(which python) src/main.py`

Initiale Einstellungen:
- `python -m venv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`

pigpio-Daemon installieren:
- `git clone https://github.com/joan2937/pigpio/`
- `cd pigpio`
- `make`
